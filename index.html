Installation

You can easily configure your app code with your theme. This documentation is to help you regarding each step of customization. Please go through the documentation carefully to understand how this template is made and how to edit this properly. HTML and CSS knowledge is not required to customize this template. You may learn basics here also.

Required Configuration

Learn how to configure some settings for Wishlist Pro App. You have to do some option manually for better user expereience. Please follow the properly. This Configuration is required.

Title Description Default
Header Menu Header menu part is manual proccess. If you want to turned on, it requires to place the code in header.liquid file. false
Collection Page Render the wishlist button on collection page then you have to do it manually. If you don't want to use the feature then you can leave it. false
Snippet Install For Installation all the snippet, you have to got to the app dashboard just click on configure app menu. After that click on install snippet. false

05. Wishlist Page Setup

You can see the preview from bellow screenshot and just copy and paste the code. Note: Please copy the code and go to the Templates folder and create a page template ls-wishlist-pro.liquid and just paste it.

Template Code Preview

                
                  <div class="page-width">
                    <div class="main__app__container" data-customer="{{ customer.id }}">
                      {% if customer.id %}
                      <div id="app__ls__mini_wishlist">
                        <div class="ls--loader--container">
                          <div class="ls--loader"></div>
                        </div>
                    	</div>
                      {% else %}
                        <div class="card">
                          <h2>You have login for access your wishlist product!</h2>
                          <a href="{{ routes.account_login_url }}" class="ls--blue--btn">LOGIN</a>
                        </div>
                    {% endif %}
                    </div>
                  </div>
                  <style>
                    .section-header {
                      margin-bottom: 25px;
                    }
                    .ls--loader--container {
                      min-height: 400px;
                      display: flex;
                      justify-content: center;
                      align-items: center;
                    }
                    .ls--loader {
                      border: 5px solid #003680;
                      border-radius: 50%;
                      border-top: 5px solid #3498db;
                      width: 40px;
                      height: 40px;
                      -webkit-animation: spin 2s linear infinite; /* Safari */
                      animation: spin 2s linear infinite;
                    }

                    /* Safari */
                    @-webkit-keyframes spin {
                      0% { -webkit-transform: rotate(0deg); }
                      100% { -webkit-transform: rotate(360deg); }
                    }

                    @keyframes spin {
                      0% { transform: rotate(0deg); }
                      100% { transform: rotate(360deg); }
                    }
                  </style>
